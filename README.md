# ``BbAppAuthProvider``

User interface components and services for authentication providers.

## Overview

The BbAppAuthProvider library provides view components and services for implementing authentication user interfaces. It includes view implementations for authentication screens, intent routing for authentication flows, sheet presentations for modal authentication, and the core service layer that coordinates authentication processes across different provider implementations.
